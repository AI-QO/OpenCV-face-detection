# OpenCV-face-detection
Detect and label faces from webcam with [OpenCV](https://opencv.org/)

## Installation Prerequisites:
Install OpenCV with `pip`
~~~
python -m pip install opencv-python
~~~
or
~~~
pip install opencv-python
~~~
